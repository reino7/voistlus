# Voistlus
Lauatennise võistluse LIVE tulemused.

**Tööks vajalik:**
> Node.JS --> https://nodejs.org/ <br />
> mongoDB --> https://www.mongodb.com/

**Käivitamiseks:**
> ```npm install```<br />
> ```node server.js```

**Väljundit vaata:**
> http://localhost:3012/index.html

**Sulgemiseks:**
> CTRL + C
