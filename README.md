# Voistlus
Lauatennise võistluse LIVE tulemused.

Tööks vajalik:
> Node.JS --> https://nodejs.org/
> mongoDB --> https://www.mongodb.com/

Käivitamiseks:
> npm init

Lisade paigaldus:
> npm install --save express socket.io mongoose

Käivitamiseks:
> node server.js

Sulgemiseks:
> CTRL + C

Väljundit vaata:
> http://localhost:3012/index.html
